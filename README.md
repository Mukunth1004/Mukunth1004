I also built a RAG document intelligence platform with pgvector, Hugging Face, and Redis achieving sub-500ms vector search with 60% reduction in API calls.

Previously at **Soft Suave Technologies** (Shopify SSR migration, PageSpeed 45→86+) and **KONE Elevator** (AWS cost prediction with XGBoost + FastAPI across 6 AWS services).
